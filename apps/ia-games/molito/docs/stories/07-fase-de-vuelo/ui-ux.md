# Story 07 — Tareas de UI/UX (Pencil)

## UX-01: Disenar estado de vuelo — Desktop

- Cuando un jugador esta en vuelo y selecciona una ficha:
  - La ficha seleccionada se resalta igual que en movimiento (glow naranja)
  - TODAS las posiciones vacias se resaltan como destinos validos (verde)
  - Indicador claro de "Fase: Vuelo" en el panel de info
  - Badge "Vuelo" junto al nombre del jugador que esta volando
- Debe ser visualmente distinguible del movimiento normal (mas destinos resaltados)

## UX-02: Disenar estado de vuelo — Mobile

- Adaptar el diseno de vuelo a mobile
- Todas las posiciones vacias como destinos deben ser claramente visibles en pantalla pequena
- Badge "Vuelo" visible junto al jugador correspondiente
